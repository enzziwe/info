# Горячие клавиши
#### Выделить всё:
Ctrl+A
#### Копировать:
Ctrl+C
#### Вставыить:
Ctrl+V
#### Сохранить:
Ctrl+S
#### Вырезать:
Ctrl+X 
#### Отменить:
Ctrl+Z
# Работа с git
#### Скачать проект:
```bush 
git clone <URL>
``` 
### Учесть  все файлы:
```bush
git add .
```
#### сохранить  коммит:
```bush
git commit -m "..."
```
### отправка в облако:
```bush
git push
```
